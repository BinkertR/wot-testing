wot-rust: Thing Description and Servient Toolbox

wot-rust provides a set of Rust crates that allow to build, consume and serve Thing Descriptions.

Usage: The wot-td crate can be used to serialize and deserialize Thing Descriptions and its extension system can be leveraged to build servients as it is done with the wot-serve extension and/or to have a type-safe support for additional ontologies.
